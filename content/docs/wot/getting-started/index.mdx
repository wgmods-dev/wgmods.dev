Welcome to the World of Tanks modding documentation!

This **Getting Started** section will help you set up your development environment and understand the core concepts you will use in every mod.

## Pre-requisite knowledge

Our documentation assumes some familiarity with programming. Before getting started, it will help if you are comfortable with:

- Python 2.7
- Basic understanding of game file structures
- ActionScript 3 (only required for UI mods)
