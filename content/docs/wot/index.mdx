## What is World of Tanks Modding?

World of Tanks modding allows you to **customize and enhance** your tank warfare experience through Python scripts, Flash components, and configuration modifications.

Mods are officially supported by Wargaming and distributed through [WGMods.net](https://wgmods.net/). All mods must comply with the [Wargaming Fair Play Policy](https://worldoftanks.eu/en/content/guide/fair_play/).

<Callout title="Open Source">
  This documentation is open source and contributions are always welcome. Feel free to open an issue or a pull request on [GitHub](https://github.com/wgmods-dev/wgmods.dev).
</Callout>

## Pre-requisite knowledge

Our documentation assumes some familiarity with programming. Before getting started, it'll help if you're comfortable with:

- **Python 2.7** (the WoT client scripting layer targets Python 2.7)
- **ActionScript 3** (required for UI mod development)
- Basic understanding of game file structures and XML

If you're new to modding, start with [Getting Started](/docs/wot/getting-started) before diving into Guides or the API Reference.

## FAQ

<Accordions>
  <Accordion id="legal-mods" title="Are mods legal in World of Tanks?">
    Yes, mods are officially supported by Wargaming. However, they must comply with the Fair Play Policy. Some mods are prohibited in ranked battles or competitive modes.

    Check the [official mod policy](https://worldoftanks.eu/en/content/guide/fair_play/) for current guidelines.
  </Accordion>
  <Accordion id="mod-safety" title="How do I know if a mod is safe?">
    Only download mods from trusted sources:

    - [WGMods.net](https://wgmods.net/) is the Official Wargaming Mod Hub
    - Open source repositories where you can review the code

    Always scan mod files for malware and check community reviews before installing.
  </Accordion>
  <Accordion id="update-compatibility" title="Do mods break with game updates?">
    Yes, most mods require updates when the game client is updated. The modding community typically releases updated versions shortly after each patch.

    Always check mod compatibility with your current game version.
  </Accordion>
</Accordions>
